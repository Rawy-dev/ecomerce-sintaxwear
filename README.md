# Sintaxwear - Tênis e Sneakers Online 👟

A **Sintaxwear** é um projeto de e-commerce moderno focado na venda de calçados estilosos e confortáveis. O site foi desenvolvido com foco em uma experiência de usuário limpa, responsiva e visualmente atraente.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

- **HTML5**: Estrutura semântica para melhor SEO e acessibilidade.
- **CSS3**: Estilização avançada com o uso de:
  - **CSS Variables**: Para fácil manutenção de cores e tipografia.
  - **Flexbox & CSS Grid**: Para layouts flexíveis e responsivos.
  - **Componentização**: Estilos divididos por componentes para melhor organização.
- **Google Fonts**: Utilização da fonte 'Ubuntu' para um design moderno.

---

## ✨ Funcionalidades

- **Design Responsivo**: Menu hamburguer funcional para dispositivos móveis.
- **Hero Section Impactante**: Banner principal com chamada para ação (CTA) clara.
- **Categorização Inteligente**: Seções dedicadas para estilos Casual, Esporte, Moderno e Futurista.
- **Grade de Produtos**: Layout dinâmico para exibição de coleções em destaque.
- **Newsletter**: Área para captura de e-mails integrada ao rodapé.
- **Redes Sociais**: Ícones vinculados às principais plataformas (Instagram, WhatsApp, TikTok, Facebook).

---

## 📁 Estrutura de Pastas

A organização do projeto segue uma estrutura modular para facilitar a escalabilidade:

```text
ecomerce-sintaxwear/
├── css/
│   ├── components/       # Estilos específicos de cada seção (Header, Footer, etc.)
│   ├── base.css          # Estilos base e botões
│   ├── reset.css         # Reset de estilos padrão do navegador
│   ├── style.css         # Importação geral (se aplicável)
│   └── variabbles.css    # Variáveis globais (cores e fontes)
├── images/
│   ├── banner/           # Imagens de fundo e heros
│   ├── categorias/       # Imagens das seções de categorias
│   ├── icones/           # SVGs e ícones de navegação
│   ├── logo/             # Logotipo oficial
│   └── produtos/         # Fotos dos produtos e grids
├── index.html            # Arquivo principal do site
└── README.md             # Documentação do projeto
```

## 🛠️ Como Executar o Projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/ecomerce-sintaxwear.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd ecomerce-sintaxwear
   ```
3. Abra o arquivo `index.html` em seu navegador ou utilize a extensão **Live Server** do VS Code para uma melhor experiência de desenvolvimento.
